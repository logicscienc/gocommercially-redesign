# 🚀 GoCommercially Homepage Redesign

A modern, conversion-focused redesign of the GoCommercially homepage built with **Next.js**, **React**, **Tailwind CSS**, and **Framer Motion**.

The goal of this redesign was to create a premium user experience with modern UI, engaging animations, and a clear marketing funnel that guides visitors from discovery to conversion.

## 🧩 Homepage Funnel

The redesigned homepage follows a marketing funnel to maximize engagement:

1. Hero Section
2. Why Choose Us
3. How It Works
4. Product Showcase
5. Testimonials
6. Frequently Asked Questions
7. Contact CTA
8. Footer

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-github-username>/gocommercially-redesign.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
