# A Studio | Design Agency Homepage

A modern and responsive Design Agency landing page built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion** as part of the **Next.js Developer Internship Task**.

---

## Live Demo

**Vercel:** https://design-agency-homepage-peach.vercel.app/

---

## GitHub Repository

**Repository:** https://github.com/logicscienc/design-agency-homepage

---

## Features

- Modern responsive design
- Sticky hero section with parallax effect
- Dark & Light mode toggle
- Animated Hero section
- Gradient typography
- Responsive navigation with mobile menu
- Services section with animated cards
- Portfolio section with auto-scrolling project showcase
- Interactive project cards linking to live demos
- Contact form with validation and success message
- Smooth scrolling navigation
- SEO metadata implementation
- Optimized images using Next/Image

---

## Tech Stack

- Next.js (App Router)
- React.js
- Tailwind CSS
- Framer Motion
- Lucide React
- React Icons
